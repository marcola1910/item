Tecnologia Utilizada

Este site ha sido creado con Apache Wicket para manejo de Pure Web utilizando POJO y patrones DDD. Tales como Service Locator, repository y root object. Bajo la capa orientada a Objetos del modelo se utiliza Hibernate mapeando los estados de los objetos a una base de datos relacional MySQL. El sistema es configurable en lo que dice respeto a motores de base de datos, la idea es agregregar implementaciones de repositorios y servicios a una base de objetos Cache.

Lo que dice respeto a control de sesiones, cada servicio cuenta con un contextmanager que mantiene la atomicidad. La logica de concurrencia utilizada es "Lo primero gana". Utilizando control de versiones de cambios evitando lock de datos y deadlocks.
